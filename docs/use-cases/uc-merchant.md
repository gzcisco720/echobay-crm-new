# 用例文档 — 商家（Merchant）

商家是**申请已获批准**的用户。他们可以登录商家后台，管理自己的品牌信息、门店、促销活动和文件。

> 注意：申请审核通过之前，商家只能看到申请状态页面，无法访问品牌/门店等功能。

---

## 登录

**入口：** `/login`

1. 打开系统，输入注册邮箱和密码
2. 点击"登录"
3. 进入商家仪表盘 `/merchant/dashboard`

**结果：** 登录成功，看到自己的数据总览

---

## UC-M1：查看仪表盘

**入口：** `/merchant/dashboard`

登录后的第一个页面，显示以下信息：

- **申请状态卡片**：当前申请是"已批准"还是"审核中"等
- **品牌摘要**：品牌名称、状态（激活/暂停/停用）
- **门店数量**：目前有几家门店
- **促销活动数量**：当前有几条促销

如果申请还没批准，仪表盘显示"您的申请正在审核中，请耐心等待"的提示。

---

## UC-M2：查看申请详情

**入口：** `/merchant/application`

1. 点击左侧菜单"申请"
2. 页面展示申请表的所有内容（只读，不可再编辑已批准的申请）
3. 如果状态是"需补充信息"，页面顶部显示管理员的备注，可以修改后重新提交

---

## UC-M3：查看品牌信息

**入口：** `/merchant/brand`

1. 点击左侧菜单"品牌"
2. 页面显示品牌的所有信息，包括：
   - 注册公司名、ABN/ACN
   - 品牌英文名/中文名
   - 品牌介绍、官网、社交媒体
   - Logo 图片（已上传的）
   - 商品类目、门店数量
   - 联系人信息
   - 当前状态（激活/暂停/停用）

**注意：** 商家目前只能查看，不能自己修改品牌信息。如需修改请联系管理员。

---

## UC-M4：查看门店列表

**入口：** `/merchant/store`（注意：这是单数，显示所有门店）

1. 点击左侧菜单"门店"
2. 列表展示所有门店，每行显示：门店名称、地址、电话、类型
3. 每行右边有"查看详情"和"编辑"按钮

**如果还没有门店：** 页面显示"暂无门店"和"添加第一家门店"的按钮

---

## UC-M5：添加新门店

**入口：** `/merchant/store` → 点"新增门店"按钮（或跳转到 Admin 创建页）

> **注意：** 根据系统设计，门店的创建和编辑功能由管理员在 Admin 后台操作（`/admin/stores/new`）。商家看到的门店列表由管理员管理。如果需要新增门店，商家应联系管理员。

---

## UC-M6：管理促销活动

**入口：** `/merchant/promotions`

### 查看促销列表

1. 点击左侧菜单"促销"
2. 列表显示所有促销活动，每条显示：
   - 促销规则描述（如"全场八折"）
   - 适用范围（品牌级 或 指定门店）
   - 开始/结束日期
   - 状态：激活 / 停用 / 计划中 / 已过期

### 新增促销

1. 点击"新增促销"按钮，跳转到 `/merchant/promotions/new`
2. 填写表单：
   - **促销规则**：描述促销内容，如"购物满 $50 返现 $5"
   - **适用范围**：整个品牌 或 选择某家门店
   - **开始日期 / 结束日期**：日期选择器
   - **排除条款**（可选）：如"不适用于已折扣商品"
   - **状态**：激活 / 停用 / 计划中
3. 点击"保存"

**结果：** 促销活动创建成功，出现在促销列表中

### 编辑促销

1. 在促销列表，点击某条促销右边的"编辑"
2. 修改对应字段
3. 点击"保存"

**结果：** 促销信息更新

### 删除促销

1. 在促销列表，点击某条促销右边的"删除"
2. 弹出确认对话框："确定要删除这条促销吗？此操作不可撤销。"
3. 点击"确认删除"

**结果：** 促销活动从列表中消失

## UC-M7：上传文件（主动上传）

**入口：** `/merchant/documents`

商家可以主动上传补充资料，也可以响应管理员的文件请求。

### 主动上传文件

1. 点击左侧菜单"文件"
2. 在"上传新文件"区域：
   - 填写文件类型说明（如"营业执照"、"保险证明"）
   - 点击"选择文件"，从电脑选择文件
   - 支持格式：PDF、Word (.doc/.docx)、Excel (.xls/.xlsx)、图片（JPG/PNG）
   - 文件大小上限：20MB
3. 点击"上传"

**结果：** 文件上传到云端，出现在"已上传文件"列表中

### 响应管理员的文件请求

1. 进入"文件"页面，顶部"待处理请求"区域显示管理员发起的请求
2. 每条请求显示：文件类型名称（如"银行对账单"）+ "上传文件"按钮
3. 点击"上传文件"，选择文件，确认上传

**结果：** 文件上传成功，管理员可以在后台看到

### 查看已上传文件

- 页面底部"全部文件"区域列出所有文件：文件名、类型、上传时间
- 点击文件名可以在新标签页打开文件

## UC-M8：忘记密码

**入口：** `/login/forgot-password`

1. 在登录页点击"忘记密码"
2. 输入注册邮箱
3. 收到重置邮件，点击链接
4. 设置新密码，登录

## 状态说明（商家视角）

### 申请状态

| 状态 | 商家看到的 | 能做什么 |
|------|-----------|---------|
| 草稿 | "尚未提交" | 可以继续填写和提交 |
| 已提交 | "等待审核" | 只能查看，不能修改 |
| 审核中 | "正在审核" | 只能查看 |
| 需补充信息 | 红色提示横幅 + 管理员备注 | 修改内容，重新提交 |
| 已批准 | "已批准" | 可以管理门店/促销/文件 |
| 已拒绝 | "未通过" | 联系管理员了解原因 |

### 品牌状态

| 状态 | 含义 |
|------|------|
| 激活 | 正常运营 |
| 停用 | 暂时关闭，不对外展示 |
| 暂停 | 管理员暂停，需联系管理员处理 |
